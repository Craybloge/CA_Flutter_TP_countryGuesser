# CA-flutter_tp_Country_guesser
 a quizz where you get information about a country and you have to guess which one it is

## Versions
Flutter 3.0.0
Dart 2.17.0

## APIS
- [drapeaux](https://www.countryflagsapi.com/)
- [data sur les pays](https://restcountries.com/)
